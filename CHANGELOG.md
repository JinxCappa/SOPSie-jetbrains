# Changelog

## [0.1.0] - Initial Release

### Added
- Encrypt and decrypt SOPS files directly from the IDE
- Decrypted preview mode for read-only viewing
- Edit-in-place with automatic re-encryption on save
- Key management: rotate data keys and update encryption keys
- Auto-detection of SOPS-encrypted files
- Custom file icons for encrypted files
- Status bar widget showing encryption state
- Editor notification banners with quick actions
- Support for `.sops.yaml` configuration files
- Configurable open/save behaviors
- Settings panel under Tools > SOPSie
