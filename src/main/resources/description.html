Seamless SOPS (Secrets OPerationS) integration for JetBrains IDEs. Edit encrypted secrets directly in your IDE without leaving your workflow.

Features

  • Encrypt & Decrypt — One-click encryption and decryption from the editor, project view, or Tools menu
  • Decrypted Preview — View decrypted content in a read-only preview without modifying the original file
  • Edit-in-Place — Edit decrypted content in a temporary file with automatic re-encryption on save
  • Key Management — Rotate data keys and update encryption keys from .sops.yaml
  • Auto-Detection — Automatically detects SOPS-encrypted files and shows encryption status
  • Visual Indicators — Custom file icons and status bar widget show encryption state at a glance
  • Editor Notifications — Smart banners with quick actions when opening encrypted files
  • Configurable Behavior — Customize open/save behaviors, auto-close tabs, confirmations, and more

Supported Formats

Works with all SOPS-supported file types: YAML, JSON, ENV, INI, and binary files.

Requirements

  • SOPS CLI installed and available in PATH (or configured in settings)
  • Encryption keys configured (AGE, GPG, AWS KMS, GCP KMS, Azure Key Vault, or HashiCorp Vault)

Getting Started

  1. Install the SOPS CLI: brew install sops (macOS) or see installation docs
  2. Configure your encryption keys
  3. Create a .sops.yaml in your project root to define encryption rules
  4. Right-click any file and select SOPS > Encrypt File to get started
